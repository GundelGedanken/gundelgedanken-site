Datenschutzerklärung

1. Datenschutz auf einen Blick

Allgemeine Hinweise

Die folgenden Hinweise geben einen einfachen Überblick darüber, was mit Ihren personenbezogenen Daten passiert, wenn Sie diese Website besuchen. Personenbezogene Daten sind alle Daten, mit denen Sie persönlich identifiziert werden können.

Datenerfassung auf dieser Website

Diese Website wird über GitHub Pages gehostet. Beim Besuch der Website werden automatisch Informationen in Server-Log-Dateien gespeichert, die Ihr Browser automatisch übermittelt. Dies sind:

  • Browsertyp und Browserversion
  • Verwendetes Betriebssystem
  • Referrer URL
  • Uhrzeit der Serveranfrage
  • IP-Adresse

Diese Daten werden von GitHub erhoben. Weitere Informationen finden Sie in der Datenschutzerklärung von GitHub.

3. Cookies & Local Storage

Diese Website verwendet keine Cookies. Es wird lediglich der Local Storage Ihres Browsers genutzt, um Ihre gewählte Darstellungspräferenz (Hell-/Dunkelmodus) zu speichern. Diese Daten verlassen Ihren Browser nicht und werden nicht an Server übermittelt.

4. Keine Tracking- oder Analyse-Tools

Diese Website verwendet keine Tracking-, Analyse- oder Marketing-Tools (kein Google Analytics, kein Matomo, keine Social-Media-Plugins). Es werden keine Nutzungsprofile erstellt.

5. Externe Links

Diese Website enthält Links zu externen Websites (z.B. X/Twitter). Beim Klick auf diese Links werden Sie auf die jeweilige externe Seite weitergeleitet. Dort gelten die Datenschutzbestimmungen des jeweiligen Anbieters.
